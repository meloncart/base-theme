Meloncart Theme
==========

![Screenshot](https://raw.githubusercontent.com/meloncart/fresh-theme/master/assets/images/theme-preview.png)

Fresh Theme is the official storefront theme for Meloncart. It provides a complete, production-ready shopping experience out of the box, built with clean HTML, CSS, and JavaScript that you can customize or use as a reference for building your own theme from scratch.

### Features
- Complete product catalog with category listings, search, and filtering
- Product detail pages with image gallery, lightbox, and variant-aware pricing
- Shopping cart with quantity controls and coupon code support
- Multi-step checkout flow (contact, shipping, payment, review)
- Single-page checkout alternative
- Guest checkout with option to create an account after purchase
- Customer account pages: login, registration, password reset, and two-factor authentication
- Profile management with address book and order history
- Digital download management for purchased files
- Wishlist and save-for-later functionality
- Product reviews and ratings display
- Shipping estimator in cart
- Responsive design
- Built with reusable partials for easy customization
- File-based templates, fully compatible with Git version control

### Requirements

Fresh Theme requires the Meloncart plugin (https://octobercms.com/plugin/meloncart-shop), October CMS 4.2+, and PHP 8.2+.
